程式整體音量 1%

調整整個程式的 Windows 音訊音量，不直接改 Twitch 網頁播放器音量。

Tampermonkey

需先將解壓縮的 Tampermonkey 放到 dist\Extensions\Tampermonkey，且第一層要有 manifest.json。WebView2 可載入擴充功能，但 Microsoft Edge 會封鎖 chrome-extension 內部管理頁；請使用 .user.js 腳本網址開啟安裝頁。請只安裝你信任的腳本。

擴充功能

把已解壓縮的擴充功能放到 dist\Extensions\擴充名稱\manifest.json。WebView2 不支援 Chrome Web Store 一鍵安裝，也不能開 chrome-extension 管理頁。

本機腳本

把 `.user.js` 或 `.js` 放到 dist\Scripts。程式會在 Twitch 原站觀看頁載入時注入腳本；不提供 Tampermonkey 的 GM_* API。
